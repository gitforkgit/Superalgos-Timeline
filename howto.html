This page is a how to, so everybody can feed this timeline/-s with Information. Just fork the repository and copy a related paragraph and insert the date(possibly acu time) and the information. First choose a meaningful headline and then a short text explaining the information.

IMPORTANT: At this point change the boxes from left to right, depending on the side the last box was on. Later on the sides could get meaning like: left side=important, right side=informative.Fell free to implement things like that or anything else as this project is meant to be for the community.
